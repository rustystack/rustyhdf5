# purehdf5-rs

A pure-Rust, zero-dependency implementation of the HDF5 file format.

## Crates

- **purehdf5-format** — Binary format parsing (`no_std` compatible)
- **purehdf5-types** — HDF5 type system
- **purehdf5-io** — I/O abstraction layer
- **purehdf5-filters** — Filter/compression pipeline
- **purehdf5-derive** — Proc macros for deriving HDF5 traits
- **purehdf5** — High-level API
